# Métodos Cuantitativos de Investigación

Este repositorio contiene **ejercicios y notebooks interactivos** para estudiantes y docentes de métodos cuantitativos de investigación.  
El objetivo es facilitar la práctica de conceptos de estadística, probabilidad, inferencia y regresión, usando ejemplos claros y ejecutables en Python (Google Colab).

## Contenido
- **Estadística descriptiva**: media, mediana, desviación estándar, gráficos.
- **Probabilidades**: eventos, distribuciones, combinatoria.
- **Regresión lineal simple**: análisis de correlación y predicción.
